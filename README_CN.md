## PCIE拆分卡图片

拆分卡接口资源介绍，支持： 

    1个PCIE插槽( x8 )

    1个M.2插槽(nvme PCIE3.0 x4)

    1个M.2插槽( 需连接至WLAN接口， PCIE3.0 x1)

    1个M.2插槽( 需连接至WWAN接口， PCIE3.0 x1)

    1个M.2插槽( 需连接至SATA接口， NGFF SATA)

    1个风扇插座(12V XH2.54)
